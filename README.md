# Curtismj1 Stakz

## How do I install these formulae?

`brew install curtismj1/stakz/<formula>`

Or `brew tap curtismj1/stakz` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "curtismj1/stakz"
brew "<formula>"
```
